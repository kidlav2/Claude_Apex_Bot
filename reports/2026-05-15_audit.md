## 4. Критические находки и следующие шаги

### 🚨 КРИТИЧНО — Orphan-позиция ENAUSDT

**Требует немедленного действия.**

Нет строки `LIVE_CLOSE` для ENAUSDT нигде в 4-дневном логе. Hard Time Stop в 19:01 UTC May 11 закрыл только ETHUSDT. Возможные варианты:

- **(a)** Bracket самоисполнился на бирже (SL ≈ TP ≈ цене входа → немедленный триггер, без PnL-записи в CSV)
- **(b)** **Позиция открыта на Binance** — Variant B (`keep winners`) мог удержать позицию, если `uPnL > 0` в момент KZ-end

**Действия:**

```bash
# 1. Проверить открытые позиции прямо сейчас
node bot.js --close-only

# 2. Или открыть Binance Futures Dashboard → Positions
```

**Технический фикс — минимальная дистанция bracket'а:**

Добавить в `binance-futures.js → placeOcoBracket()` проверку до размещения:

```javascript
const MIN_BRACKET_TICKS = 5;
const ticker = await publicRequest("/fapi/v1/ticker/price", { symbol });
const markNow = numRequired(ticker.price, "markPrice");

if (Math.abs(parseFloat(tpTrigger) - markNow) < MIN_BRACKET_TICKS * filters.tickSize)
  throw new Error(`TP ${tpTrigger} within ${MIN_BRACKET_TICKS} ticks of mark — degenerate bracket`);

if (Math.abs(parseFloat(slTrigger) - markNow) < MIN_BRACKET_TICKS * filters.tickSize)
  throw new Error(`SL ${slTrigger} within ${MIN_BRACKET_TICKS} ticks of mark — degenerate bracket`);
```
